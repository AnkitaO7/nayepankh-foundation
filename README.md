# NayePankh Foundation Website

A simple responsive NGO website created as part of a frontend internship selection task for NayePankh Foundation using HTML, CSS, and JavaScript.

## Features

Responsive design for mobile and desktop 
Dark mode toggle feature 
Hero section with background image 
Multiple sections (Home, About, Gallery, Contact) 
Image gallery grid layout 
Interactive button with popup message 
Social media links in footer

## Technologies Used

HTML 
CSS 
JavaScript

## Project Purpose

This project was built as part of a frontend internship selection task.
It demonstrates basic UI design, responsive layout, and simple JavaScript interactivity.
